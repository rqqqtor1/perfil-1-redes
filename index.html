Tarea de Investigación 1: Componentes de Red

1. ¿Qué son los componentes pasivos en la Red?

Los componentes pasivos en una red son aquellos que no requieren energía eléctrica para funcionar. Su función principal es conducir, conectar o encaminar señales, pero no las amplifican ni las procesan.

2. Descripción técnica de cada componente pasivo

RJ45

El RJ45 es un conector físico usado comúnmente en las redes Ethernet. Es el punto de conexión para los cables de red UTP que se utilizan para transmitir datos.

3. ¿Qué son los componentes activos en la Red?

Los componentes activos son dispositivos que requieren energía eléctrica para funcionar y son responsables de procesar y amplificar las señales en una red.

4. Descripción técnica de cada componente activo

Switch

El switch es un dispositivo de red que conecta múltiples dispositivos dentro de una red local (LAN) y utiliza la conmutación de paquetes para enviar datos solo al dispositivo de destino.

5. Herramientas utilizadas para realizar cableado estructurado

Ponchadora

La ponchadora es una herramienta utilizada para insertar y asegurar los cables en un conector, como un conector RJ45, al cable de red.

6. ¿Qué es el cableado horizontal?

El cableado horizontal es la parte del sistema de cableado estructurado que conecta los puntos de distribución (puntos de red) con las estaciones de trabajo en una misma planta o piso.

7. ¿Qué es el cableado vertical o backbone?

El cableado vertical o backbone es el sistema de cables que interconecta los diferentes pisos de un edificio o áreas dentro de una misma red, usualmente conectando los dispositivos de distribución central con los de distribución intermedia.

8. Tipos de cables para el sistema de cableado horizontal

Cable UTP Cat5e

El cable UTP Cat5e es un tipo de cable utilizado en redes Ethernet que soporta velocidades de hasta 1 Gbps. Es el estándar mínimo para la mayoría de las instalaciones de red.

9. Normativa para hacer cables de red (Clase A y B)

La normativa de cables de red especifica el orden en el que deben ser conectados los hilos de colores dentro de un conector RJ45. Existen dos estándares principales para la terminación de cables de red: T568A y T568B. Ambos esquemas determinan el orden de los cables para asegurarse de que la red funcione correctamente y de que los cables estén conectados de manera uniforme en toda la instalación.

  • T568A: Este estándar es más común en aplicaciones gubernamentales y residenciales. Es preferido en nuevas instalaciones porque es compatible con ambas configuraciones, A y B.
  • T568B: Es el estándar más utilizado en Estados Unidos para instalaciones comerciales y en redes que necesitan ser compatibles con sistemas telefónicos antiguos.

10. Para qué se utiliza un cable directo en la red

El cable directo se utiliza para conectar dispositivos diferentes entre sí dentro de una red. Por ejemplo, se utiliza para conectar una computadora a un switch, un router, o a un punto de acceso. En un cable directo, los hilos en ambos extremos están conectados en el mismo orden, lo que permite que los dispositivos se comuniquen de manera eficiente.

11. Para qué se utiliza un cable cruzado en la red

El cable cruzado se utiliza principalmente para conectar dispositivos similares directamente entre sí, como conectar dos computadoras directamente sin la necesidad de un switch o router intermedio. En un cable cruzado, dos pares de cables están cruzados en uno de los extremos, lo que invierte las señales de transmisión y recepción, permitiendo la comunicación directa.
